# Changelog

All notable changes to the "Vocalize MD" extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

## [0.2.0] - 2026-01-13

### Added
- Click-to-jump functionality: click any sentence to jump to that position in audio
- Hover states on sentences to indicate they're clickable

### Changed
- Completely redesigned panel UI with modern, fluid aesthetic
- New deep green color scheme (#0f2e24)
- Premium glassmorphism controls with backdrop blur
- Animated pulsing status indicator
- Subtle vinyl disc animation behind play button
- Enhanced play button with layered shadows and ring effect
- Gradient dividers and ambient glow effects
- Smooth fade masks on text area edges
- Active sentence highlighting with slide animation
- Refined typography and thinner scrollbar
- Smaller logo in README

### Fixed
- Updated GitHub releases link in README

## [0.1.0] - 2026-01-12

### Added
- Initial release of Vocalize MD extension
- Text-to-speech conversion for Markdown files using Deepgram TTS
- Intelligent Markdown processing using Gemini AI via OpenRouter
- Support for multiple Deepgram voice models (Aura voices)
- Real-time word-by-word highlighting during playback
- Webview panel for audio playback controls
- Configurable API keys for Deepgram and OpenRouter
- Configurable voice selection
- Automatic text chunking for long documents
- Error handling with user-friendly messages
- MIT license
- Comprehensive documentation (README, CONTRIBUTING)
- Project icon and branding
